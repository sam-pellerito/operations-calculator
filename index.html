How the math works (one-time)

B = S * r (marketing budget from savings)
ΔR = B * mROI (additional revenue)
ΔC = ΔR / AOV (additional customers)
incremental CPA ≈ AOV / mROI
current customers C0 = R0 / AOV
new avg CPA = (C0 * currentCPA + B) / (C0 + ΔC)
ΔContribution = ΔR * cm
new margin % ≈ (R0*M0 + ΔContribution) / (R0 + ΔR)

Quick guidance

  • If incremental CPA (AOV/mROI) < current CPA, new customers are cheaper than today's average.
